# ✈️ NBA Charter Flight Tracker

A real-time flight tracking bot that monitors Delta Air Lines charter flights and cross-references the NBA schedule to identify which teams are traveling to away games — delivered as Discord notifications.

Built as a self-directed Python project to develop scripting, API integration, and data pipeline skills.

---

## How It Works

1. **Polls FlightRadar24** every 10 minutes for all active Delta Air Lines flights
2. **Identifies charter flights** by callsign pattern — NBA charters operate as `DAL88xx` and `DAL89xx` flight numbers
3. **Resolves airports to arenas** using geodesic distance — finds the nearest NBA arena to each flight's origin and destination airport
4. **Cross-references the NBA schedule** to confirm a game is scheduled within 48 hours
5. **Sends a Discord notification** with flight details, team matchup, and game date

```
FR24 API ──► Charter Detection ──► Arena Matching ──► Schedule Check ──► Discord
```

## Setup

### 1. Clone the repo

```bash
git clone https://github.com/yourusername/nba-charter-tracker.git
cd nba-charter-tracker
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure environment variables

```bash
cp .env.example .env
```

Edit `.env` with your credentials:

```
DISCORD_TOKEN=your_discord_bot_token
DISCORD_CHANNEL_ID=your_channel_id
FR24_USERNAME=your_fr24_email
FR24_PASSWORD=your_fr24_password
```

### 4. Fetch the current NBA schedule

```bash
python schedule_fetcher.py
```

### 5. Run the bot

```bash
python tracker.py
```

3. Add environment variables in Railway's dashboard (Settings → Variables)
4. Railway will detect the `Procfile` and run the bot automatically

## How I Discovered the Charter Pattern

Delta operates NBA charter flights under `DAL88xx` and `DAL89xx` callsigns — four-digit flight numbers in the 8800–8999 range. This is a common convention across airlines for charter/non-revenue operations. I identified the pattern by observing Delta flights on FlightRadar24 on game days and cross-referencing with known team travel schedules.
